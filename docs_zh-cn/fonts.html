概述

M.U.G.E.N 1.0中的字体由带有"def"扩展名的定义文件和随附的sff或True Type字体文件组成。字体的这种格式称为FNT v2。

在M.U.G.E.N 1.0之前的版本中，字体表示为扩展名为"fnt"的单个二进制文件。此格式称为FNT v1。 M.U.G.E.N 1.0与两种字体格式兼容。

本文档介绍FNT v2字体的结构和格式。

术语

Character
单个字母，数字或符号。
Glyph
单个字符的可视化。
Bitmap font
一种字体，其字形完全由精灵表示。位图字体的受支持集中的每个字符都映射到一个精灵，该精灵是该字符的字形。
TrueType font
一种字体，其字形由描述如何呈现每个字形的矢量表示。 TrueType字体通常在Microsoft Windows和其他操作系统中用于为用户界面呈现抗锯齿（平滑）字体。
Color bank
位图字体的调色板，该调色板将颜色方案应用于字形。字体可能包含一个或多个颜色库。
Color bank number
标识特定颜色库的正数。

FNT v2 def 文件

FNT v2格式的字体的必需组成部分是字体定义文件，该文件是扩展名为"def"的文本文件。字体可能伴随着与def文件位于同一目录中的第二个文件。该文件是什么取决于字体的类型。

FNT v2字体可以是位图字体或TrueType字体。

位图字体

带有内嵌注释的位图字体定义文件示例如下所示。

[FNT v2]
; FNT v2 version number.  Don't change this.
fntversion = 2,00
; Name of this font.
name = "4x6 variable"
; This font contains the full ASCII character set.
; All letters are uppercase.

[Def]
; This is a bitmap font
Type = bitmap
; Size of font: width, height.  Width is used for spaces.
Size = 3,6
; Spacing between font glyphs: width, height.
Spacing = 1,0
; Drawing offset: x, y.
Offset = 0,1
; Filename of the sff containing the glyphs.  Use sff v2 only.
File = f-4x6.sff

; Note: All units are in pixels.
; Text rendered with bitmap fonts may be in ASCII only.
; See work/font/f-4x6/f-4x6.def for information on creating the sff.

位图字体仅支持ASCII字符集中的字符。不支持扩展的ASCII集。有关ASCII字符集的表格，请参见下面的"ASCII字符参考"。

位图字体必须附带一个SFF文件，其中包含字形的精灵。字体中的每个字符都映射到一个以0，<ASCII码>索引的精灵，其中<ASCII码>是该字符的相应ASCII码。例如，字符‘A’（ASCII代码65）映射为sprite 0,65。

没有相应精灵的任何字符都不会被渲染。

位图字体可能包含一个或多个颜色库。每个色库映射到存储在SFF中的调色板，并以0，<色库号>索引，其中<色库号>是正数。如果在字体渲染期间指定了无效的颜色库，则将使用SFF中第一个精灵的调色板。

TrueType字体

带有内嵌注释的示例TrueType字体定义文件如下所示：

[FNT v2]
; FNT v2 version number.  Don't change this.
fntversion = 2,00
; Name of this font.
name = "Arial"
; Font author
author = "Microsoft"

[Def]
; This is a truetype font.
Type = truetype
; Size of font: width, height.  Only height is used for truetype fonts.
Size = 18,36
; Spacing between font glyphs.  Only height is used for truetype fonts.
Spacing = 1,1
; Drawing offset: x, y.
Offset = 0,0
; Filename of the font to load.  Will search Windows font directory if not in current directory.
File = arial.ttf
; Preferred blending mode: 0 - none, 1 - blended.
Blend = 1

; Note: All units are in pixels.
; Text rendered with truetype fonts may be ASCII or UTF-8

TrueType字体定义必须随附TrueType字体文件，该文件的扩展名通常为.ttf或.ttc。如果TrueType字体文件与定义文件位于同一目录中，则将加载它。否则，M.U.G.E.N将尝试从Windows字体目录中加载字体文件。
